# Prod Prototypes
Work Product Prototypes built to address different challenges faced by different organisations. All prototypes are functional with mock data, and are WCAG 2.2 A11y compliant. Built using Claude Code in combination with Lovable.

## nucao: 
###[v1: DTC Lab](https://nucao-growth-lab.lovable.app)
###[v2: Pathfinder](https://nucao-growth-navigator.lovable.app)
Showcasing the DTC growth accelerator opportunities for nucao. Includes funnel comparison, cohort dashboard, UTM attribution & fallback.

## Lovable: [LovGov](https://lovgov.lovable.app)
Governance Gateway streamlines enterprise app deployment by automating IT approval workflows.

## Kleinanzeigen: [Steward Beacon](https://stewedbacon.lovable.app/) 
Steward Beacon is an Outcome Copilot providing sellers with AI-driven insights to understand and optimize their listing performance.

## 1KOMMA5°: [CardioPulse](https://cardiopulse.lovable.app/)
Customer Hub intended to close the post-sale gap, providing transparency from contract to live in five visible steps.

## Carbmee: [CarbAuditMe](https://carbauditme.lovable.app)
A confidence metadata layer for Carbmee's supplier module: quality scores, validation depth, and BOM coverage that travel with every CSRD and CBAM filing.

## Workflex: 
### a) [GeoFlex](https://geoflex.lovable.app/)
GeoFlex Mobility Insights provides strategic insights for global workforce mobility programs.
### b) Case Study: [PreFlex](https://preflex.lovable.app/)
An immersive product case study prototype for pre trip compliance scoring.

## EMMA: [Hypnos](https://emmahypnos.lovable.app/)
Tiered Configuration Governance, managing customer care platform configurations.

## PVcase: [SolarFinLens](https://solarfinlens.lovable.app)
Financial Lens converts solar yield data into investment-grade financial metrics.

* no direct affiliation with any of these organisations or associated entities, prototypes are built explicitly for demonstrative purposes.
